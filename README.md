# Strata

### The task was to develop a _responsive_ portfolio site based on [HTML5UP](https://html5up.net/) design but on pure HTML and CSS.

- has a contact form
- implemented gallery
- contains animations
- contains modal windows
- developed first for desktop, then for other devices
